MARKET IS THE PROJECTS NAME
>>> from market import db

CREATE ALL THE DATABASES, THIS WILL CREATE A FILE CALLED SITE
>>> db.create_all()

IMPORT THE DATABASE MODULES
>>> from market import User, Post

CREATE AN OBJECT OF USER
>>> user_1 = User(username='Seba', email='testing@demo.com', password='password')

ADD THE USER TO THE DB
>>> db.session.add(user_1)
>>> user_2 = User(username='JohnDoe', email='ing@demo.com', password='password')
>>> db.session.add(user_2)

COMMIT THE CHANGES SO THEY ACTUALLY ARE IN THE DB
>>> db.session.commit()

RETURN ALL THE MODULES WITH THEIR RESPECTIVE INFO
>>> User.query.all()
[User('Seba', 'testing@demo.com', 'default.jpg'), User('JohnDoe', 'ing@demo.com', 'default.jpg')]

RETURN THE FIRST INSTANCE OF THE MODULE
>>> User.query.first()
User('Seba', 'testing@demo.com', 'default.jpg')

RETURNS All INSTANCES WITH THE SAME NAME OF THE MODULE (AS A LIST)
>>> User.query.filter_by(username='Corey').all()
[]
>>> User.query.filter_by(username='Seba').all()
[User('Seba', 'testing@demo.com', 'default.jpg')]

SAVE THE INSTANCE INTO A VARIABLE
>>> user = User.query.filter_by(username='Seba').all()
>>> user
[User('Seba', 'testing@demo.com', 'default.jpg')]

IN ORDER TO GET JUST ONE INSTANCE INSTEAD OF A LIST, SPECIFY THE FIRST FOUND
>>> User.query.filter_by(username='Seba').first()
User('Seba', 'testing@demo.com', 'default.jpg')
>>> user = User.query.filter_by(username='Seba').first()
>>> user
User('Seba', 'testing@demo.com', 'default.jpg')
